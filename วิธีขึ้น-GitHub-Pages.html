🚀 เอาแดชบอร์ดขึ้นเว็บ — ได้ลิงก์ถาวร

ทำเอง 5 ขั้น ประมาณ 2 นาที · คุณล็อกอิน GitHub และมี repo hrbwp02-beep/Sale พร้อมแล้ว

1
ดาวน์โหลดไฟล์ index.html
กดการ์ดดาวน์โหลด "index.html" ที่ผมส่งให้ในแชต (ข้อความก่อนหน้านี้) เซฟไว้ในเครื่อง จำให้ได้ว่าเซฟไว้โฟลเดอร์ไหน (ปกติอยู่ใน Downloads)
2
เปิดหน้า repo แล้วกด Add file → Upload files
ไปที่ github.com/hrbwp02-beep/Sale แล้วมองมุมขวาบน จะเห็นปุ่มเขียว Add file
github.com/hrbwp02-beep/Sale
Code ▾ ＋ Add file ▾
▾ เมนูจะเด้งลงมา → เลือก Upload files
👉 กดปุ่ม "Add file" (กรอบแดง) แล้วเลือก "Upload files"
3
ลากไฟล์ index.html เข้าไปวาง แล้ว Commit
ลากไฟล์ที่โหลดมา มาวางในกรอบ แล้วกดปุ่มเขียว Commit changes ด้านล่าง
Sale · Upload files
⬆️ ลากไฟล์ index.html มาวางตรงนี้
(หรือกด "choose your files" เพื่อเลือกไฟล์)
Commit changes
👉 หลังลากไฟล์เข้าแล้ว กดปุ่มเขียว "Commit changes"
4
เปิด GitHub Pages: Settings → Pages
ในหน้า repo กดแท็บ Settings (บนขวา) → เมนูซ้ายเลื่อนหา Pages → ตั้งค่า Branch เป็น main และโฟลเดอร์ / (root) → กด Save
Sale · Settings · Pages
Build and deployment — Source: Deploy from a branch
🌿 Branch: main ▾ 📁 / (root) ▾ Save
👉 เลือก main + /(root) แล้วกด Save
5
รอ ~1 นาที → ได้ลิงก์ถาวร 🎉
รีเฟรชหน้า Pages อีกที จะมีลิงก์เขียว ๆ ขึ้นมา — นี่คือลิงก์ถาวรที่ส่งให้ใครก็ได้:
https://hrbwp02-beep.github.io/Sale/
✅
