## 3. หน้าจอทั้ง 9 หน้า

1. **Sales Overview** — ภาพรวมผู้บริหาร: KPI หลัก 5 ตัว (มูลค่าขายรวม, ปริมาณขายรวม, ราคาเฉลี่ย/Kg, จำนวนลูกค้า, จำนวนผลิตภัณฑ์) + กราฟแนวโน้ม + insight เด่น
2. **Sales Overview (รายเดือน)** — ตารางยอดขายรายเดือน เทียบ 2569 vs 2568 (มูลค่า · ปริมาณ · ราคาเฉลี่ย)
3. **Product Analysis** — วิเคราะห์รายสินค้า 5 ประเภท + การเติบโต MoM + จัดอันดับ (Product Ranking)
4. **Customer Analysis** — วิเคราะห์ลูกค้า 90 ราย, Top 10, ส่วนแบ่ง %, การเติบโตรายเดือน
5. **Customer Contribution** — Pareto Analysis: ลูกค้ากี่รายสร้างยอดขายกี่ % (กฎ 80/20)
6. **Product Mix** — สัดส่วนยอดขายแต่ละกลุ่มสินค้า (Donut chart)
7. **Price Analysis** — ราคาขายเฉลี่ย (฿/Kg) แนวโน้มรายเดือน
8. **Year Comparison** — เปรียบเทียบยอดขาย 2568 · 2569 เคียงข้างกัน
9. **Forecast** — คาดการณ์ยอดขายสิ้นปี (AI Forecasting) พร้อมระดับความเชื่อมั่น

## 4. การอัปเดตข้อมูล (Data Editor)

เปิดหน้า `/bwp-data-editor` — แก้ตัวเลขแล้วเว็บจริงเปลี่ยนทันทีที่รีเฟรช **โดยไม่ต้อง deploy ใหม่**

**ขั้นตอน:**
1. เปิดหน้า Data Editor
2. แก้ไขค่าในตาราง:
   - **KPI หลัก** — ตัวเลขการ์ดด้านบนสุด (label, ค่า, หน่วย)
   - **ยอดขายรายเดือน ปี 2569** — เติมเดือนใหม่ตรงช่องว่าง (มูลค่า / ปริมาณ)
3. กด **💾 บันทึกเข้า Supabase**
4. กลับมาหน้าแดชบอร์ด กด **↻ โหลดใหม่** หรือรีเฟรช (Ctrl+Shift+R)

> ปุ่ม "บันทึก" จะกดได้เมื่อมีการแก้ไขข้อมูลแล้วเท่านั้น
> ข้อมูลถูกเก็บที่ Supabase (cloud) — ทุกคนที่เปิดเว็บจะเห็นค่าใหม่ทันที

---

## 5. แหล่งข้อมูล & การ Deploy

- **ข้อมูลสด:** ดึงจาก Supabase ทุกครั้งที่เปิดเว็บ (ตาราง `dashboard_data`)
- **ข้อมูลสำรอง:** ถ้าต่อ Supabase ไม่ได้ จะใช้ข้อมูลที่ฝังในไฟล์ (`data.js`) แทนอัตโนมัติ
- **การ Deploy:** ทุกครั้งที่ push โค้ดขึ้น branch `main` → Vercel จะ build + เผยแพร่ใหม่อัตโนมัติภายใน ~1 นาที (ผ่าน Git Integration)

## 6. แก้ปัญหาเบื้องต้น

| อาการ | วิธีแก้ |
|-------|---------|
| เห็นข้อมูลเก่า | กด Ctrl+Shift+R ล้าง cache |
| หน้าจอว่าง/ดำ | กด F12 ดู Console แล้วแจ้งทีมพัฒนา |
| แก้ข้อมูลแล้วไม่เปลี่ยน | ตรวจว่ากด "บันทึกเข้า Supabase" แล้ว → รีเฟรชหน้าแดชบอร์ด |
| ตัวเลขไม่ตรง | เปิด Data Editor ตรวจค่าที่บันทึกล่าสุด |
